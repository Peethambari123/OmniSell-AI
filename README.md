# Agentic AI Conversational Retail System

## Overview
This Streamlit application demonstrates an agentic AI-based conversational
sales platform for omnichannel retail (ABFRL use case).

## Features
- Multi-agent orchestration
- Conversational shopping journey
- Loyalty & payment simulation
- Modular, extensible architecture

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
